物质的量是高中化学计算的核心，串联起宏观与微观世界。本专题涵盖5大考

考点4：平均摩尔质量

M平 = m总 / n总 = (M1·n1 + M2·n2) / (n1+n2)

对气体：M平 = M1·φ1 + M2·φ2（φ为体积分数）

相对密度法：M平 = D × M参（D为相对密度，M参为参考气体摩

尔质量）

考点5：十字交叉法

十字交叉法原理

用于两组分混合物的快速计算。设两组分 A、B 的某物理量分别为 a、b，混合后为 c，则：

nA / nB = (c − b) / (a − c)

适用场景：平均摩尔质量、平均相对原子质量、溶液混合浓度计算等。

二、典例破题

【典例1】NA 陷阱综合

NA 为阿伏伽德罗常数的值。下列说法正确的是（ ）

A. 标准状况下，22.4 L H2O 含有的分子数为 NA

B. 1 mol Na2O2 与足量 CO2 充分反应，转移电子数为 NA

C. 常温常压下，22.4 L N2 含有的分子数为 NA

D. 1 L 1 mol/L Na2CO3 溶液中含 CO32− 数为 NA

【答案】B

A × 标况下 H2O 为液态，不能用 22.4 L/mol。

B √ 2Na2O2+2CO2=2Na2CO3+O2，Na2O2 中 O 为 −1 价，发生歧化：一半升为

0 价（O2），一半降为 −2 价（Na2CO3）。每 2 mol Na2O2 转移 2 mol e−，故 1

mol Na2O2 转移 1 mol e−，即 NA。

C × 常温常压不是标况，22.4 L 气体物质的量 ≠ 1 mol。

D × CO32− 水解：CO32−+H2O⇌HCO3−+OH−，故 CO32− 数 < NA。

【典例2】溶液配制误差分析

配制 100 mL 0.10 mol/L Na2CO3 溶液，下列操作会使浓度偏低的是（ ）

A. 用托盘天平称量时，药品与砝码放反（使用游码）

B. 定容时俯视刻度线

C. 未用蒸馏水洗涤烧杯内壁

D. 容量瓶使用前用 0.10 mol/L Na2CO3 溶液润洗

【答案】AC

A √ 偏低 左物右码放反后，实际称取质量 = 砝码质量 − 游码质量，偏小，n 偏小，

c 偏低。

B × 偏高 俯视使液面未达刻度线就已停止，V 偏小，c 偏高。

C √ 偏低 烧杯内壁残留溶质未转移，n 偏小，c 偏低。

D × 偏高 润洗使容量瓶内壁残留溶质，n 偏大，c 偏高。

三、巩固提升

考点1 巩固题（n/N/NA/m/M/V/Vm/c 关系）

Q1-1

0.5 mol H2SO4 中含 H 原子的物质的量为多少？含 O 原子的个数为多少？（NA 表示）

【答案与解析】

1 个 H2SO4 含 2 个 H，4 个 O。

n(H) = 0.5 mol × 2 = 1 mol

N(O) = 0.5 mol × 4 × NA = 2NA

Q1-2

标准状况下，11.2 L CO2 的质量是多少？含有多少个原子？

【答案与解析】

n(CO2) = 11.2 L / 22.4 L/mol = 0.5 mol

m(CO2) = 0.5 mol × 44 g/mol = 22 g

1 个 CO2 含 3 个原子（1C+2O），N(原子) = 0.5 × 3 × NA = 1.5NA

Q1-3

将 5.85 g NaCl 溶于水配成 500 mL 溶液，求溶液中 Na+ 的物质的量浓度。

【答案与解析】

M(NaCl) = 23 + 35.5 = 58.5 g/mol

n(NaCl) = 5.85 g / 58.5 g/mol = 0.1 mol

c(NaCl) = 0.1 mol / 0.5 L = 0.2 mol/L

NaCl = Na+ + Cl−，c(Na+) = c(NaCl) = 0.2 mol/L

考点2 巩固题（NA 陷阱）

Q1-4

判断正误：标准状况下，22.4 L 己烷（C6H14）含有的共价键数目为 19NA。

【答案与解析】

错误。标况下己烷为液态（含 6 个碳的烷烃在标况下为液体），不能用 22.4 L/mol

计算。即使强行计算，22.4 L 也不等于 1 mol。

Q1-5

NA 为阿伏伽德罗常数的值。判断：1 mol Fe 与足量稀 HNO3 反应，转移电子数为 3NA。

【答案与解析】

不一定。若稀 HNO3 过量，Fe 被氧化为 Fe3+，转移 3NA e−；若 Fe 过量，可能生

成 Fe2+（Fe+2Fe3+=3Fe2+），需结合具体条件分析。此题表述不严谨，按"足量稀

HNO3"应认为氧化为 Fe3+，转移 3NA，正确。

Q1-6

判断：1 L 0.1 mol/L NaHCO3 溶液中 HCO3− 和 CO32− 离子数之和为 0.1NA。

【答案与解析】

错误。根据物料守恒，n(HCO3−) + n(CO32−) + n(H2CO3) = 0.1 mol。HCO3− 既

可电离（HCO3−⇌H++CO32−）也可水解（HCO3−+H2O⇌H2CO3+OH−），故

HCO3−+CO32− < 0.1NA。

考点3 巩固题（溶液配制）

Q1-7

配制 250 mL 0.20 mol/L NaOH 溶液，需要称取 NaOH 固体多少克？

【答案与解析】

n(NaOH) = 0.20 mol/L × 0.250 L = 0.05 mol

M(NaOH) = 23 + 16 + 1 = 40 g/mol

m(NaOH) = 0.05 mol × 40 g/mol = 2.0 g

Q1-8

配制一定物质的量浓度溶液时，下列操作对浓度有何影响（填"偏大""偏小""无影响"）：

(1) 溶解后未冷却即转移至容量瓶中 ______

(2) 定容时液面超过刻度线，用胶头滴管吸出多余部分 ______

(3) 转移时有少量溶液溅出 ______

【答案与解析】

(1) 偏大——热溶液体积膨胀，冷却后液面下降，实际 V 偏小，c 偏大。

(2) 偏小——吸出多余部分时也吸走了溶质，n 偏小，c 偏小。

(3) 偏小——溶质损失，n 偏小，c 偏小。

Q1-9

实验室需用 480 mL 0.10 mol/L 的 CuSO4 溶液，应选用容量瓶的规格和称取胆矾

（CuSO4·5H2O）的质量分别是？

【答案与解析】

容量瓶规格只有 500 mL，没有 480 mL，应选用 500 mL 容量瓶。

n(CuSO4) = 0.10 mol/L × 0.500 L = 0.05 mol

M(CuSO4·5H2O) = 64 + 32 + 16×4 + 5×18 = 250 g/mol

m = 0.05 mol × 250 g/mol = 12.5 g

考点4 巩固题（平均摩尔质量）

Q1-10

某混合气体由 CH4 和 C2H6 组成，其对 H2 的相对密度为 12.5，求 CH4 与 C2H6 的物质的量

之比。

【答案与解析】

M平 = D × M(H2) = 12.5 × 2 = 25 g/mol

M(CH4)=16，M(C2H6)=30

设 n(CH4)=x，n(C2H6)=y

16x + 30y = 25(x+y) → 16x+30y = 25x+25y → 5y = 9x → x : y = 5 : 9

n(CH4) : n(C2H6) = 5 : 9

Q1-11

标准状况下，测得 0.50 g 某气体的体积为 0.28 L，求该气体的摩尔质量。

【答案与解析】

n = V / Vm = 0.28 L / 22.4 L/mol = 0.0125 mol

M = m / n = 0.50 g / 0.0125 mol = 40 g/mol

Q1-12

CO 和 CO2 的混合气体 18 g，完全燃烧后测得 CO2 体积为 11.2 L（标况），求原混合气体

的平均摩尔质量。

【答案与解析】

燃烧反应：2CO + O2 = 2CO2

最终 CO2 总量 n = 11.2 / 22.4 = 0.5 mol

设原 CO 为 x mol，CO2 为 y mol，则 x + y = 0.5（碳守恒），28x + 44y =

解：28x + 44(0.5−x) = 18 → 28x + 22 − 44x = 18 → −16x = −4 → x =

0.25, y = 0.25

M平 = 18 / 0.5 = 36 g/mol

Q1-14

O2 和 O3 的混合气体中氧元素的质量分数为 90%，求 O2 与 O3 的物质的量之比。

【答案与解析】

O2 中 O%=100%，O3 中 O%=100%——两者都是纯氧元素，质量分数均为

100%，无法用十字交叉法根据氧元素质量分数区分。

正确思路：设 O2 x mol，O3 y mol，总质量 = 32x + 48y，氧元素质量 = 32x +

48y

此数据不足以求解。题目可能想考察的是平均摩尔质量：M平 = 32x+48y/(x+y)。

此题数据有误，不适用十字交叉法。

（说明：此题旨在让学生意识到十字交叉法需满足"可加和性"前提。）

Q1-15

已知空气中 N2 和 O2 的体积比约为 4 : 1，求空气的平均摩尔质量。

【答案与解析】

对气体，体积比 = 物质的量比。

M平 = (4×28 + 1×32) / 5 = (112+32)/5 = 144/5 = 28.8 g/mol

（通常取 29 g/mol 作为空气的平均摩尔质量。）

四、易错警示

⚠ 专题01 易错点警示

标况条件：必须同时满足 0°C 和 101 kPa，只提"标准状况"才可用 Vm=22.4 L/mol。

物质状态：标况下 H2O、SO3、CCl4、酒精、苯、碳原子数≥5的烷烃均为非气态。

粒子种类：注意问的是"原子数"还是"分子数"，是"质子数"还是"电子数"。

容量瓶规格：常见规格为 50 mL、100 mL、250 mL、500 mL、1000 mL，配制时选

择"大于且最接近"所需体积的规格。

定容操作：距刻度线 1~2 cm 处必须改用胶头滴管，不能直接倾倒。

十字交叉法注意：仅适用于两组分混合、可加和的物理量（如质量、物质的量、摩尔质

量），不适用于体积（非理想混合）或不可加和的量。

二、典例破题

【典例1】氧化还原概念辨析

在反应 3Cl2 + 6KOH = 5KCl + KClO3 + 3H2O 中，被氧化的氯原子与被还原的氯原子的

物质的量之比为（ ）

A. 1:5   B. 5:1   C. 1:3   D. 3:1

【答案】A

Cl2 中 Cl 为 0 价。产物中 KCl 的 Cl 为 −1 价（被还原），KClO3 的 Cl 为 +5 价（被

氧化）。

每生成 1 个 KClO3（1 个 Cl 被氧化），同时生成 5 个 KCl（5 个 Cl 被还原）。

被氧化 : 被还原 = 1 : 5，选 A。

【典例2】化合价升降法配平——详细步骤

配平：KMnO4 + HCl → KCl + MnCl2 + Cl2↑ + H2O

【答案与配平步骤】

第一步：标化合价

KMnO4：K(+1)，Mn(+7)，O(−2)；HCl：H(+1)，Cl(−1)

产物：MnCl2：Mn(+2)；Cl2：Cl(0)

第二步：确定变价元素 Mn: +7→+2（降5），Cl: −1→0（升1）

第三步：最小公倍数 5 和 1 的最小公倍数 = 5

KMnO4 系数 = 5/5 = 1，被氧化的 Cl 原子数 = 5/1 = 5

第四步：初步配平

2KMnO4 + 10HCl(被氧化) + ?HCl(起酸作用) → 2KCl + 2MnCl2 + 5Cl2↑ + ?

H2O

实际上，KMnO4 系数应为 2（因为右边有 2 个 MnCl2 需要和 2 个 KCl 对应 2 个

K）：

2KMnO4 + 16HCl → 2KCl + 2MnCl2 + 5Cl2↑ + 8H2O

第五步：检查

K: 2=2 √；Mn: 2=2 √；O: 8=8 √；H: 16=16 √；Cl: 16=2+4+10=16 √

最终：2KMnO4 + 16HCl = 2KCl + 2MnCl2 + 5Cl2↑ + 8H2O

【典例3】陌生方程式配平

配平离子方程式：MnO4− + Fe2+ + H+ → Mn2+ + Fe3+ + H2O

【答案与步骤】

Mn: +7→+2，降5；Fe: +2→+3，升1

【典例2】电子式正误判断

下列电子式书写正确的是（ ）

（H在上方）

【答案】A

解释：为什么 H2O 的沸点（100°C）比 H2S 的沸点（−60°C）高得多？

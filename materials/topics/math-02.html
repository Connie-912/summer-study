专题02：解三角形
知识精讲
定理/公式表达式适用场景
正弦定理a/sinA=b/sinB=c/sinC=2R已知两角一边（AAS/ASA）；已知两边一对角（SSA）
余弦定理a2b2c2=+−2bc·cosA已知两边夹角（SAS）；已知三边（SSS）
三角形面积S=½ab·sinC=½bc·sinA=½ac·sinB已知两边及其夹角
海伦公式√[p(p−a)(p−b)(p−c)]，S=p=(a+b+c)/2已知三边长度
边角互化sinA:sinB:sinC=a:b:c利用正弦定理实现边与角的正弦之间的转化
考点1：正弦定理  
典例破题

例1：在 ΔABC 中，A = 60°，B = 45°，a = 6，求 b。

解：由正弦定理 a/sinA = b/sinB，得 6/sin60° = b/sin45°，∴ b = 6× sin45° / sin60° = 6 × (√2/2) / (√3/2) = 6√2/√3 = 2√6。

巩固1

在 ΔABC 中，a = 10，A = 30°，B = 120°，求 b。

答案：b = 10√3。由正弦定理：10/sin30° = b/sin120°，∴ b = 10 ×sin120° / sin30° = 10 × (√3/2) / (1/2) = 10√3。

巩固2

在 ΔABC 中，A = 75°，B = 45°，b = 8，求 a（sin75° = (√6+√2)/4）。

答案：a = 4(√3+1)。由正弦定理：a/sin75° = 8/sin45°，∴ a = 8 × sin75° /sin45° = 8 × ((√6+√2)/4) / (√2/2) = 8 × (√6+√2)/4 × 2/√2 =4(√6+√2)/√2 = 4(√3+1)。

巩固3

在 ΔABC 中，a = 5√3，b = 5，A = 60°，求 B。

答案：B = 30°。由正弦定理：5√3/sin60° = 5/sinB，得 5√3/(√3/2) = 5/sinB，即 10 = 5/sinB，∴ sinB = 1/2。又 a > b，∴ A > B，故 B =30°。

易错警示

· SSA 情况（已知两边及其中一边的对角）可能有两解、一解或无解，必须判断。

· 正弦定理中 a/sinA = 2R（R为外接圆半径），不是 a = sinA。

典例破题
·角度单位必须统一，不能混用度与弧度。

 考点5：综合应用
 典例破题例1：在中，a=2√3，b=2，B=30°，求和c。ΔABCA

解：由正弦定理 sinA = a·sinB/b = 2√3×sin30°/2 = √3/2，∴A=60° 或 120°。若 A=60° 则 C=90°，c = a·sinC/sinA = 2√3×1/(√3/2) = 4；若 A=120° 则 C=30°，c = 2√3×sin30°/sin120° =2。本题有两解。

巩固1

在 ΔABC 中，a=2，b=3，c=4，求 cosA + cosB + cosC。

答案：21/16。cosA = (b2+c2−a2)/(2bc) = (9+16−4)/(2×3×4) = 21/24 =7/8；cosB = (a2+c2−b2)/(2ac) = (4+16−9)/(2×2×4) = 11/16；

cosC = (a2+b2−c2)/(2ab) = (4+9−16)/(2×2×3) = −3/12 = −1/4。

通分求和：7/8 + 11/16 − 1/4 = 14/16 + 11/16 − 4/16 = 21/16。

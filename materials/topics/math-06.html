预习说明

本专题为高二上学期内容预习。空间向量是平面向量的自然延伸，将二维拓展到三维。建议先复习专题01的平面向量基础，再学习本专题。

知识精讲
知识模块核心内容与平面向量的联系
空间向量运算加法、减法、数乘、数量积；坐标表示(x,y,z)与平面向量运算规则完全一致，多一个维度
线面位置关系线线平行/垂直、线面平行/垂直、面面平行/垂直用向量方向（方向向量、法向量）来判定
空间角计算异面直线夹角、线面角、二面角夹角公式|a·b|/(|a||cosθ=b|)
考点1：空间向量运算  
典例破题

例1：已知 a = (1, 2, 3)，b = (−1, 0, 2)，求 a·b 和 |a|。

解：a·b = 1×(−1) + 2×0 + 3×2 = −1+0+6 = 5。|a| = √(12+22+32) =√(1+4+9) = √14。

巩固1

已知 a = (2, −1, 3)，b = (1, 4, −2)，求 a + 2b。

答案：(4, 7, −1)。a+2b = (2, −1, 3) + 2(1, 4, −2) = (2, −1, 3) + (2, 8, −4) = (4, 7,−1)。

巩固2

已知 a = (1, 1, 0)，b = (0, 1, 1)，求 a 与 b 的夹角余弦值。

答案：cosθ = 1/2。|a| = √2，|b| = √2，a·b = 1×0+1×1+0×1 = 1。cosθ= 1/(√2×√2) = 1/2，∴ θ = 60°。

巩固3

已知 A(1,0,0), B(0,1,0), C(0,0,1)，求向量 AB 和 AC 的数量积。

答案：1。AB = (−1, 1, 0)，AC = (−1, 0, 1)。AB·AC = (−1)×(−1) + 1×0+ 0×1 = 1。

易错警示

· 空间向量坐标有三个分量，运算时不要漏掉 z 分量。

模长公式：|a| = √(x2+y2+z2)，分母是平方和再开根。·

· 两个向量垂直的充要条件是数量积为 0（与平面向量一致）。

考点2：线面位置关系判定
典例破题

例1：直线 l 的方向向量为 s = (1, 2, −1)，平面 α 的法向量为 n = (2, −1, 0)，判断 l 与 α 的位置关系。

解：s·n = 1×2 + 2×(−1) + (−1)×0 = 2−2+0 = 0，∴ s ⊥ n。方向向量与法向量垂直，说明直线与平面平行或在平面内。

巩固1

判断向量 a = (2, −4, 6) 与 b = (−1, 2, −3) 是否平行。

答案：平行。a = −2b，即 (2, −4, 6) = −2(−1, 2, −3)，两向量共线。

巩固2

平面 α 过点 (1,0,0) 且法向量为 n = (1,1,1)，写出平面的点法式方程。

答案：x + y + z − 1 = 0。1×(x−1) + 1×(y−0) + 1×(z−0) = 0，即x−1+y+z = 0，∴ x+y+z = 1。

巩固3

判断直线 l（方向向量 s=(1,1,0)）与平面 α（法向量 n=(1,−1,2)）是否垂直。

答案：不垂直。直线垂直平面 ⇔ 方向向量平行于法向量。s 与 n 不平行（1/1 ≠1/−1），∴ 直线不垂直于平面。另外 s·n = 1−1+0 = 0，说明直线平行于平面。

易错警示

· 直线方向向量与平面法向量垂直 ⇒ 直线平行于平面（或含于平面内）。

· 直线方向向量与平面法向量平行 ⇒ 直线垂直于平面。

· 两个平面法向量平行 ⇒ 两平面平行；法向量垂直 ⇒ 两平面垂直。

考点3：空间角计算
典例破题

例1：求向量 a = (1, 0, 0) 与 b = (1, 1, 1) 的夹角。

解：cosθ = (a·b)/(|a||b|) = (1×1+0×1+0×1)/(1×√3) = 1/√3 =√3/3。∴ θ = arccos(√3/3)。

巩固1

求向量 a = (2, 0, 0) 与 b = (0, 3, 0) 的夹角。

答案：90°。cosθ = (2×0+0×3+0×0)/(2×3) = 0，∴ θ = 90°。

巩固2

已知 A(1,0,0), B(0,1,0), C(0,0,1)，求 cos∠BAC。

答案：1/2。AB = (−1, 1, 0)，AC = (−1, 0, 1)。|AB| = √2，|AC| = √2，AB·AC = 1。cos∠BAC = 1/(√2×√2) = 1/2，∴ ∠BAC = 60°。

巩固3

直线 l 的方向向量为 s = (1, 1, 0)，平面 α 的法向量为 n = (1, 0, 1)，求 l 与 α所成角的正弦值。

答案：sinθ = 1/2。直线与平面所成角 θ 满足 sinθ = |s·n|/(|s||n|) = |1+0+0|/(√2×√2) = 1/2，∴ θ = 30°。

易错警示
· 异面直线的夹角取锐角（或直角），公式中加绝对值：cosθ = |a·b|/(|

a||b|)。

· 线面角的正弦值：sinθ = |s·n|/(|s||n|)，其中 s 为方向向量，n 为法向量。

· 二面角的平面角等于两平面法向量的夹角（或其补角）。

预习提示：静电场是高二上学期的重要内容，也是高中物理的难点之

一。它与力学中的重力场有类比关系，建议在理解力学的基础上进行预习。以下内容为入门级，重点建立基本概念。

⚡考点1：库仑定律 
库仑定律：F = k Q₁Q₂ / r²
静电力常量：k = 9.0 × 109 N·m²/C²

与万有引力的类比：形式完全相同，都是平方反比律！

方向：同种电荷相斥，异种电荷相吸
类比：库仑力 F = kQ1Q2/r2 与万有引力 F = Gm1m2/r2 形式相

同。区别在于：万有引力只有引力，库仑力有引力和斥力；k 远大于 G（静电力 >> 万有引力）。

①【巩固题】

两个点电荷，电荷量分别为+2 μC和-3 μC，相距0.1 m。求它们之间的库仑力的大小和方向。（k=9.0×109）

详细解析

F = k Q₁Q₂ / r²F = 9.0×109 × (2×10-6) × (3×10-6) / (0.1)²

 9.0×1096×10-12×/0.01F=54×10-3/0.01=F=5.4N方向：异种电荷，相互吸引。  
 【巩固题】2两个完全相同的小球，分别带电荷量+q和-q，接触后再分开到原来距离。求前后库仑力之比。  
 详细解析接触前：F₁kq²/r²（引力）=接触后：两球完全相同，电荷中和后均分。总电荷q+(-q)=0，每个球带电0。F₂=0前后力之比：F₁0（不再有力的作用）:F₂=1:注意：若两球带同种电荷，接触后每个球带(q₁+q₂)/2的电荷量。  
③【巩固题】

比较两个电子之间的库仑力与万有引力的大小。已知电子质量m=9.1×10-31 kg，电子电荷量e=1.6×10-19 C，G=6.67×10-11，k=9.0×109。

详细解析

F电 / F引 = (ke²/r²) / (Gm²/r²) = ke² / (Gm²)= 9.0×109 × (1.6×10-19)² / [6.67×10-11 × (9.1×10-31)²]= 9.0×109 × 2.56×10-38 / (6.67×10-11 × 8.28×10-61)

≈ 2.30×10-28 / 5.52×10-71 ≈ 4.2 × 1042

库仑力是万有引力的约1042倍！微观粒子间万有引力完全可以忽略。

📖考点2：电场强度 
电场强度定义：E = F / q（检验电荷所受电场力与电荷量

之比）

点电荷的电场：E = kQ / r²
方向：正电荷受电场力方向与E相同，负电荷相反
电场叠加：E合 = E₁ + E₂ + ...（矢量叠加）
与重力场的类比：重力场强度 g = G/m（即重力加速度），电场

强度 E = F/q。两者都是描述场的性质的物理量，与放入场中的物体（检验电荷）无关。

①【巩固题】

在电场中某点放入电荷量为2×10-8 C的检验电荷，受到4×10-4N的电场力。求该点的电场强度。若改放电荷量为-4×10-8 C的检验电荷，该点电场强度是多少？该电荷受力多大？

详细解析
电场强度：E = F/q = 4×10-4 / (2×10-8) = 2×104 N/C

电场强度是场的性质，与检验电荷无关，所以改放后 E 仍为 2×104 N/C。

 (-4×10-8)(2×104)-8×10-4受力：F×==qE=N负号表示力方向与电场方向相反。 
 【巩固题】2两个点电荷Q₁=+4×10-8C和Q₂=-4×10-8C相距0.2m。求它们连线中点的电场强度。（k=9.0×109） 
详细解析

中点距两电荷均为 r = 0.1 m。

Q₁在中点产生的电场：E₁ = kQ₁/r² = 9×109×4×10-8/0.01 =

3.6×104 N/C（方向背离Q₁，指向Q₂）
Q₂在中点产生的电场：E₂ = k|Q₂|/r² = 同样大小 = 3.6×104 N/
C（方向指向Q₂，即与E₁同向）
E合 = E₁ + E₂ = 7.2×104 N/C（方向由Q₁指向Q₂）
③【巩固题】

关于电场强度，下列说法正确的是：A. 由E=F/q可知，E与F成正比 B. E的方向与检验电荷受力方向相同 C. 电场中某点不放检验电荷时E=0   D. E是描述电场本身性质的物理量，与检验电荷无关

详细解析

A. 错误，E由场源电荷决定，与检验电荷的F和q无关。E=F/q是定义式，不是决定式。

B. 错误，E的方向是正电荷受力方向，负电荷受力方向与E相反。

C. 错误，电场强度是场的固有属性，与是否放入检验电荷无关。

D. 正确。

答案：D
📖考点3：电势能与电势差 
电势能：Ep = qφ（类比重力势能 Ep = mgh）
电势：φ = Ep/q（类比重力势中的"高度"h）
电势差（电压）：UAB = φA - φB = WAB/q
电场力做功：WAB = qUAB = -ΔEp

 【巩固题】2在匀强电场中，将电荷量为q的正电荷沿电场线方向移动距离d，电场强度为E。求：(1)电场力做的功；(2)电势能的变化量。  
 详细解析(1)电场力做功：FqE，沿电场线方向移动=qEd（正功）W=Fd=(2)电势能变化：ΔEp=-W=-qEd电势能减少了qEd（类比重力场中物体下降，重力势能减少）。注意：正电荷沿电场线方向移动，电势能减小；负电荷沿电场线方向移动，电势能增大。  
③【巩固题】

关于电势和电势能，下列说法正确的是：A. 电势高的地方电势能一定大 B. 正电荷在电势高的地方电势能大 C. 电势为零的地方电势能一定为零 D. 电势能是标量，没有正负之分

详细解析

A. 错误，电势能 Ep = qφ，与电荷正负有关。负电荷在电势高的地方电势能反而小。

B. 正确，正电荷 q > 0，Ep = qφ，φ越大Ep越大。

C. 正确（电势零点处，Ep=q×0=0），但电势零点是人为选取的，不是客观存在的物理事实。通常题目中C选项的判断需要结合

具体语境。

D. 错误，电势能是标量，但有正负之分（正电荷在正电势处电势能为正，在负电势处电势能为负）。

综合来看，B是最准确无误的选项。

答案：B

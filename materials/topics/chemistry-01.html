专题01 · 物质的量

物质的量是高中化学计算的核心，串联起宏观与微观世界。本专题涵盖5大考点，基础薄弱者务必从概念关系网络图入手，逐步建立"物质的量中心"思维。

一、知识精讲
考点1：n / N / NA / m / M / V / Vm / c 的关系网络
核心关系网络（以物质的量 n 为中心）
÷NA ↓ ↑ ×NAN（粒子数）
n（物质的量）×M↓↑÷Mm（质量）
÷Vm ↓ ↑ ×VmV（气体体积）
×Vaq ↓ ↑ ÷Vaqn（物质的量）
c（物质的量浓度）

n = N / NA   |   n = m / M   |   n = V / Vm（标况 Vm=22.4 L/mol）  |   n = c · VaqNA ≈ 6.02 × 1023 mol−1

考点3：一定物质的量浓度溶液的配制与误差分析
配制步骤（以配制 100 mL 1.00 mol/L NaCl 溶液为例）

计算 → 称量 → 溶解 → 转移（至100 mL容量瓶）→ 洗涤（烧杯和玻璃棒2~3次）→ 定容（距刻度线1~2 cm改用胶头滴管）→ 摇匀 → 装瓶贴签

操作的变化c原因
未洗涤烧杯和玻璃棒偏小溶质损失，n减小
定容时俯视刻度线偏大实际偏小V
定容时仰视刻度线偏小实际偏大V
定容摇匀后液面下降再加水偏小偏大V
容量瓶未干燥无影响定容时仍加水至刻度
砝码生锈偏大称量质量偏大，n偏大
考点4：平均摩尔质量  

M平 = m总 / n总 = (M1·n1 + M2·n2) / (n1+n2)对气体：M平 = M1·φ1 + M2·φ2（φ为体积分数）相对密度法：M平 = D × M参（D为相对密度，M参为参考气体摩尔质量）

【答案与解析】

容量瓶规格只有 500 mL，没有 480 mL，应选用 500 mL 容量瓶。

n(CuSO4) = 0.10 mol/L × 0.500 L = 0.05 molM(CuSO4·5H2O) = 64 + 32 + 16×4 + 5×18 = 250 g/molm = 0.05 mol × 250 g/mol = 12.5 g

【答案与解析】

O2 中 O%=100%，O3 中 O%=100%——两者都是纯氧元素，质量分数均为100%，无法用十字交叉法根据氧元素质量分数区分。

正确思路：设 O2 x mol，O3 y mol，总质量 = 32x + 48y，氧元素质量 = 32x +48y

    ⚠专题01易错点警示·标况条件：必须同时满足0°C和101kPa，只提"标准状况"才可用L/mol。Vm=22.4·物质状态：标况下H2O、SO3、CCl4、酒精、苯、碳原子数≥5的烷烃均为非气态。·粒子种类：注意问的是"原子数"还是"分子数"，是"质子数"还是"电子数"。·容量瓶规格：常见规格为50mL，配制时选mL、100mL、250mL、500mL、1000择"大于且最接近"所需体积的规格。·定容操作：距刻度线1~2处必须改用胶头滴管，不能直接倾倒。cm·十字交叉法注意：仅适用于两组分混合、可加和的物理量（如质量、物质的量、摩尔质量），不适用于体积（非理想混合）或不可加和的量。

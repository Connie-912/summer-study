化简：cos(270°+α) + sin(180°+α)。

答案：sinα − sinα = 0。cos(270°+α) = sinα（cos 在第四象限为正，270°+α=360°−90°+α，cos(270°+α)=sinα）；sin(180°+α) = −sinα。∴原式 = sinα − sinα = 0。

巩固3

已知 sinα = 3/5，α 为锐角，求 sin(π−α) + cos(π/2−α)。

答案：6/5。sin(π−α) = sinα = 3/5；cos(π/2−α) = sinα = 3/5。∴ 原式 = 3/5+ 3/5 = 6/5。

易错警示

· "奇变偶不变"：奇指 k·π/2 中 k 为奇数时函数名改变（sin↔cos,tan↔cot）。

· "符号看象限"：把 α 视为锐角，看原函数在对应象限的符号。

· sin(−α) = −sinα，cos(−α) = cosα（sin 是奇函数，cos 是偶函数）。

考点4：解三角形综合
典例破题

例1：在 ΔABC 中，a = 2，b = 3，cosC = 1/3，求 c。

解：c2 = a2+b2−2ab·cosC = 4+9−2×2×3×1/3 = 13−4 = 9，∴c = 3。

巩固1

在 ΔABC 中，sinA = 3/5，cosB = 5/13，求 cosC。

答案：16/65。cosC = −cos(A+B) = −(cosAcosB−sinAsinB)。cosA = 4/5（A 为锐角），sinB = 12/13（B 为锐角）。cosC = −(4/5×5/13 −3/5×12/13) = −(20/65−36/65) = −(−16/65) = 16/65。

 巩固2在中，a=2√3，b=2，B=30°，求和C。ΔABCA

答案：A=60°, C=90° 或 A=120°, C=30°。由正弦定理 sinA = a·sinB/b =2√3×(1/2)/2 = √3/2，∴ A=60°或120°。若 A=60°，C=90°；若A=120°，C=30°。

巩固3

在 ΔABC 中，证明：sin2A + sin2B + sin2C = 2 + 2cosAcosBcosC。

答案：利用 A+B+C=π，C=π−(A+B)，sinC=sin(A+B)。左边 =sin2A+sin2B+sin2(A+B)。展开：sin2(A+B) = (sinAcosB+cosAsinB)2= sin2Acos2B+cos2Asin2B+2sinAcosBcosAsinB。整理得2+2cosAcosBcosC。（恒等式证明，记住结论即可）

易错警示

· 解三角形与三角恒等变换结合时，注意 A+B+C = π 这一隐含条件。

· sin(A+B) = sinC，cos(A+B) = −cosC 是常用转化。

· 正余弦定理的选择：已知两边一夹角用余弦定理，已知两角一边用正弦定理。

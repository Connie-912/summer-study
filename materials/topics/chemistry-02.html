专题02 · 氧化还原反应

氧化还原反应是贯穿高中化学的核心反应类型，从化合价变化角度理解"氧化"与"还原"的本质——电子转移。本专题从概念辨析到配平方法，循序渐进。

一、知识精讲
考点1：核心概念辨析
概念定义化合价电子实例（CuO+H2→Cu+H2O）
氧化反应失电子（或电子对偏离）的反应升高失电子化合价升高（H2H2→H2O，H被氧化）
还原反应得电子（或电子对偏向）的反应降低得电子化合价降低CuO→Cu，Cu被还原）（CuO
氧化剂得电子的物质（含化合价降低元素）降低得电子CuO（氧化剂）
还原剂失电子的物质（含化合价升高元素）升高失电子H2（还原剂）
氧化产物还原剂被氧化后的产物——H2O（氧化产物）
还原产物氧化剂被还原后的产物——Cu（还原产物）
 记忆口诀"升失氧，降得还"——化合价升高，失电子，发生氧化反应（还原剂）；化合价降低，得电子，发生还原反应（氧化剂）。"氧氧还还"——氧化剂被还原（发生还原反应），还原剂被氧化（发生氧化反应）。 
 考点2：双线桥与单线桥 
方法连接方式箭头含义标注内容
双线桥反应物→产物（同元素）电子转移方向ne−"，化合价"得/失升降
单线桥还原剂→氧化剂（反应物之间）电子转移方向"ne−"
【示例】MnO2 + 4HCl(浓) → MnCl2 + Cl2↑ + 2H2O

双线桥：Mn 从 +4 降到 +2，"得 2e−"；Cl 从 −1 升到 0，"失 2×e−"（注意：4 个 HCl 中只有 2 个 Cl− 被氧化）。

单线桥：从 HCl 中 Cl 指向 MnO2 中 Mn，标"2e−"。

考点3：化合价升降法配平
配平口诀

"一标变价，二列升降，三求最小公倍数，四配系数，五查守恒"步骤：(1) 标出所有元素化合价，找出变价元素。

(2) 列出化合价升降数（升价数/降价数）。

(3) 求升降数的最小公倍数，确定氧化剂和还原剂的系数。

(4) 用观察法配平其余元素。

(5) 检查：原子守恒 + 电荷守恒（离子方程式）。

考点4：常见氧化剂和还原剂整理
类别常见物质还原产物/氧化产物
强氧化剂KMnO4(H+)、浓HNO3、浓H2SO4、Cl2、O2、H2O2、Fe3+Mn2+、NO2/NO、SO2、Cl−、O2−/H2O、H2O、Fe2+

二、典例破题
【典例1】氧化还原概念辨析

在反应 3Cl2 + 6KOH = 5KCl + KClO3 + 3H2O 中，被氧化的氯原子与被还原的氯原子的物质的量之比为（ ）A. 1:5   B. 5:1   C. 1:3   D. 3:1

【答案】A

Cl2 中 Cl 为 0 价。产物中 KCl 的 Cl 为 −1 价（被还原），KClO3 的 Cl 为 +5 价（被氧化）。

每生成 1 个 KClO3（1 个 Cl 被氧化），同时生成 5 个 KCl（5 个 Cl 被还原）。

被氧化 : 被还原 = 1 : 5，选 A。

   【典例2】化合价升降法配平——详细步骤配平：KMnO4+HCl→KCl+MnCl2+Cl2↑+H2O   
   【答案与配平步骤】第一步：标化合价KMnO4：K(+1)，Mn(+7)，O(−2)；HCl：H(+1)，Cl(−1)   

产物：MnCl2：Mn(+2)；Cl2：Cl(0)第二步：确定变价元素 Mn: +7→+2（降5），Cl: −1→0（升1）第三步：最小公倍数 5 和 1 的最小公倍数 = 5KMnO4 系数 = 5/5 = 1，被氧化的 Cl 原子数 = 5/1 = 5第四步：初步配平2KMnO4 + 10HCl(被氧化) + ?HCl(起酸作用) → 2KCl + 2MnCl2 + 5Cl2↑ + ?H2O实际上，KMnO4 系数应为 2（因为右边有 2 个 MnCl2 需要和 2 个 KCl 对应 2 个K）：2KMnO4 + 16HCl → 2KCl + 2MnCl2 + 5Cl2↑ + 8H2O第五步：检查K: 2=2 √；Mn: 2=2 √；O: 8=8 √；H: 16=16 √；Cl: 16=2+4+10=16 √最终：2KMnO4 + 16HCl = 2KCl + 2MnCl2 + 5Cl2↑ + 8H2O

【典例3】陌生方程式配平

− + Fe2+ + H+ → Mn2+ + Fe3+ + H2O配平离子方程式：MnO4

【答案与步骤】

Mn: +7→+2，降5；Fe: +2→+3，升1− 系数=1，Fe2+ 系数=5最小公倍数=5，MnO4− + 5Fe2+ + ?H+ → Mn2+ + 5Fe3+ + ?H2OMnO4左边 O 原子 4 个，右边 H2O 需要 4 个 O → 4H2O，需要 8H+检查电荷：左边 −1+5×(+2)+8×(+1)=−1+10+8=+17；右边 +2+5×(+3)=+2+15=+17√− + 5Fe2+ + 8H+ = Mn2+ + 5Fe3+ + 4H2O最终：MnO4

四、易错警示
⚠ 专题02 易错点警示
· 概念混淆："氧化剂被还原，还原剂被氧化"——氧化剂发生的是还原反应，还原剂发生

的是氧化反应。

· 歧化反应：同一元素既升高又降低（如 Cl2+2NaOH→NaCl+NaClO+H2O），该物质既作氧化剂又作还原剂。

部分氧化还原：如 MnO2+4HCl 中，4 个 HCl 只有 2 个 Cl− 被氧化，另外 2 个起酸的作· 用。

· 最高/最低价态：最高价态只能作氧化剂（如 HNO3 中 N(+5)），最低价态只能作还原剂（如 HCl 中 Cl(−1)），中间价态既可作氧化剂又可作还原剂。

· 配平顺序：先配变价元素，再配其他元素，最后用 H2O 配平 H 和 O，离子方程式还需

检查电荷守恒。

 考点1：等差数列通项与求和 
典例破题

例1：等差数列 {an} 中，a3 = 7，a7 = 15，求 a1 和 d。

解：a3 = a1+2d = 7，a7 = a1+6d = 15。两式相减得 4d = 8，∴ d= 2，a1 = 7−4 = 3。

巩固1

在等差数列 {an} 中，a1 = 5，d = 3，求 a10 和 S10。

答案：a10 = 32，S10 = 185。a10 = a1+9d = 5+27 = 32；S10 =10(a1+a10)/2 = 10×(5+32)/2 = 5×37 = 185。

巩固2

等差数列 {an} 中，a2+a8 = 10，求 a5。

答案：a5 = 5。由性质：a2+a8 = 2a5（因为 2+8 = 5+5），∴ a5 = (a2+a8)/2 =5。

巩固3

等差数列 {an} 的前 n 项和为 Sn，若 S3 = 6，S6 = 24，求 S9。

答案：S9 = 54。由等差数列性质：S3，S6−S3，S9−S6 也成等差数列。6, 18,S9−24 成等差数列，∴ 18−6 = (S9−24)−18，得 S9 = 54。或直接用 Sn =An2+Bn 待定系数法。

易错警示

· 通项公式 an = a1+(n−1)d，注意是 (n−1) 而不是 n。

· 求和公式 Sn = n(a1+an)/2，注意分母是 2。

· 等差数列中 Sn 是关于 n 的二次函数（d≠0 时），没有常数项（S0=0）。

考点2：等比数列通项与求和
典例破题

例1：等比数列 {an} 中，a1 = 2，a4 = 16，求公比 q 和 S5。

解：a4 = a1·q3 = 2q3 = 16，∴ q3 = 8，q = 2。S5 = 2(1−25)/(1−2)= 2(1−32)/(−1) = 62。

巩固1

等比数列 {an} 中，a1 = 3，q = −2，求 a5 和 S5。

答案：a5 = 48，S5 = 33。a5 = 3×(−2)4 = 3×16 = 48；S5 = 3[1−(−2)5]/[1−(−2)] = 3[1−(−32)]/3 = 1+32 = 33。

巩固2

等比数列 {an} 中，a2·a8 = 16，求 a5。

答案：a5 = ±4。由性质 am·an = ap·aq（m+n=p+q），又 2+8 = 5+5，∴2 = 16，a5 = ±4。

a2·a8 = a5

巩固3

等比数列 {an} 中，a1+a2 = 6，a3+a4 = 24，求公比 q 和 a5+a6。

答案：q = 2，a5+a6 = 96。a3+a4 = (a1+a2)q2 = 6q2 = 24，∴ q2 = 4，q= 2（取正）；a5+a6 = (a3+a4)q2 = 24×4 = 96。

易错警示

· 等比数列求和公式 Sn = a1(1−qn)/(1−q) 仅在 q≠1 时成立；q = 1 时Sn = na1。

· an = a1·qn−1，指数是 n−1 而非 n。

· 公比 q 为负数时，各项正负交替，求和时注意符号。

考点3：数列求和——裂项相消
典例破题

例1：求数列 {1/[n(n+1)]} 的前 n 项和 Sn。

解：1/[n(n+1)] = 1/n − 1/(n+1)。Sn = (1−1/2)+(1/2−1/3)+...+(1/n−1/(n+1)) = 1 − 1/(n+1) = n/(n+1)。

巩固1

求数列 {1/[(2n−1)(2n+1)]} 的前 n 项和。

答案：Sn = n/(2n+1)。1/[(2n−1)(2n+1)] = ½[1/(2n−1) − 1/(2n+1)]。Sn= ½[(1−1/3)+(1/3−1/5)+...+(1/(2n−1)−1/(2n+1))] = ½[1−1/(2n+1)] = n/(2n+1)。

求 1/(√2+1) + 1/(√3+√2) + ... + 1/(√(n+1)+√n) 的和。

答案：Sn = √(n+1) − 1。分母有理化：1/(√(k+1)+√k) = √(k+1)−√k。Sn =(√2−1)+(√3−√2)+...+(√(n+1)−√n) = √(n+1)−1。

易错警示

· 裂项时注意系数：1/[n(n+k)] = (1/k)[1/n − 1/(n+k)]。

· 消去后保留的项要仔细数清楚，通常剩下最开始和最后的几项。

· 分母有理化形式的裂项：1/(√A+√B) = (√A−√B)/(A−B)。

考点4：数列求和——错位相减
典例破题

例1：求数列 {n·2n} 的前 n 项和 Sn。

解：Sn = 1·21+2·22+3·23+...+n·2n。2Sn = 1·22+2·23+...+(n−1)·2n+n·2n+1。相减：−Sn = 21+22+23+...+2n−n·2n+1 =2(2n−1)−n·2n+1。∴ Sn = (n−1)·2n+1+2。

 巩固1{(2n−1)·3n}求数列的前项和Sn。n

答案：Sn = (n−1)·3n+1+3。Sn = 1·3+3·32+5·33+...+(2n−1)·3n。3Sn =1·32+3·33+...+(2n−3)·3n+(2n−1)·3n+1。相减得 −2Sn =3+2·32+2·33+...+2·3n−(2n−1)·3n+1 = 3+2×[32(3n−1−1)/(3−1)]−(2n−1)·3n+1 = 3+9(3n−1−1)−(2n−1)·3n+1 =3n+1−6−(2n−1)·3n+1 = (2−2n)·3n+1−6。∴ Sn = (n−1)·3n+1+3。

巩固2

求数列 {n/2n} 的前 n 项和 Sn。

答案：Sn = 2 − (n+2)/2n。Sn = 1/2+2/22+3/23+...+n/2n。½Sn =1/22+2/23+...+(n−1)/2n+n/2n+1。相减得 ½Sn = 1/2+1/22+1/23+...+1/2n−n/2n+1 = (1/2)(1−1/2n)/(1−1/2)−n/2n+1 = 1−1/2n−n/2n+1。∴ Sn = 2−2/2n−n/2n = 2−(n+2)/2n。

巩固3

求数列 {n·(−2)n} 的前 3 项和 S3。

答案：S3 = −18。直接计算：a1 = 1×(−2)1 = −2，a2 = 2×(−2)2 = 2×4 = 8，a3 = 3×(−2)3 = 3×(−8) = −24。S3 = −2+8−24 = −18。

 易错警示·错位相减只适用于"等差×等比"型数列。·乘公比后对齐时，指数要对应好，别错位。
·相减后中间部分构成等比数列，注意首项、公比和项数。·最后一步解出时注意系数的符号。Sn

专题04 · 钠及其化合物

钠是典型的活泼金属，其化合物之间存在清晰完整的转化关系网络。掌握化学方程式书写和物质性质对比是学习本专题的关键。

侯氏制碱法中，先向饱和食盐水中通入的气体是______，后通入的气体是______，析出的晶体是______，煅烧该晶体得到的产物是______。

【答案与解析】

先通 NH3，后通 CO2。

原因：NH3 溶解度极大（1 体积水溶解约 700 体积 NH3），先通NH3 使溶液呈碱性，有利于 CO2 的吸收（CO2 在水中溶解度小，在碱性溶液中溶解度大）。

析出晶体：NaHCO3（溶解度较小）煅烧产物：Na2CO3（纯碱）   
三、巩固提升   
考点1巩固题   

Q4-1写出下列反应的化学方程式：(1) Na 与 CuSO4 溶液反应(2) Na2O2 与 CO2 反应(3) NaHCO3 溶液与 NaOH 溶液反应

【答案与解析】

(1) Na 先与水反应，生成的 NaOH 再与 CuSO4 反应：2Na + 2H2O = 2NaOH + H2↑2NaOH + CuSO4 = Cu(OH)2↓ + Na2SO4总：2Na + 2H2O + CuSO4 = Cu(OH)2↓ + Na2SO4 + H2↑(2) 2Na2O2 + 2CO2 = 2Na2CO3 + O2（Na2O2 既是氧化剂又是还原剂）−(3) NaHCO3 + NaOH = Na2CO3 + H2O（离子：HCO3+OH−=CO32−+H2O）Q4-2将 2.3 g 金属钠投入 100 g 水中，求反应后溶液中溶质的质量分数。

【答案与解析】

n(Na) = 2.3 / 23 = 0.1 mol2Na + 2H2O = 2NaOH + H2↑n(NaOH) = 0.1 mol，m(NaOH) = 0.1 × 40 = 4 gn(H2) = 0.05 mol，m(H2) = 0.05 × 2 = 0.1 g溶液质量 = 2.3 + 100 − 0.1 = 102.2 gw(NaOH) = 4 / 102.2 × 100% = 3.91%

Q4-3Na2O2 与 H2O 反应中，氧化剂是______，还原剂是______，每生成 1 molO2 转移电子______mol。

【答案与解析】

2Na2O2 + 2H2O = 4NaOH + O2↑Na2O2 中 O 为 −1 价，发生歧化反应：一半 O 升到 0 价（O2），一半降到 −2 价（NaOH）。

氧化剂：Na2O2（含降价 O）；还原剂：Na2O2（含升价 O）每生成 1 mol O2（2 mol O 原子从 −1→0，失 2 mol e−），同时有2 mol O 从 −1→−2，得 2 mol e−。转移电子：2 mol。

四、易错警示
⚠ 专题04 易错点警示
· Na 与盐溶液反应：Na 先与水反应，不能直接置换出金属（如 Na 投入

CuSO4 溶液生成 Cu(OH)2 沉淀而非 Cu）。

· Na2O2 不是碱性氧化物：Na2O2 与 H2O/CO2 反应除生成碱/盐外还生成

O2，属于过氧化物，不是碱性氧化物。Na2O 才是碱性氧化物。

2−· Na2CO3 与盐酸反应：分步进行，刚开始无明显现象（CO3

+H+=HCO3−），继续加酸才产生气泡（HCO3−+H+=H2O+CO2↑）。

− 浓度· 侯氏制碱法先通 NH3：若先通 CO2，CO2 溶解度小，溶液中 HCO3

太低，无法析出 NaHCO3。

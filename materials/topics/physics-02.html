⚖考点1：力的合成与分解 
核心知识：力是矢量，合成与分解遵循平行四边形定则。两个力

F₁、F₂夹角为θ，合力 F = √(F₁² + F₂² + 2F₁F₂cosθ)。分力一般按效果分解（如沿斜面方向和垂直斜面方向）。

夹角θ合力大小方向
0°F₂（最大）F=F₁+与F₁、F₂方向相同
90°F=√(F₁²+F₂²)tanα=F₂/F₁
180°F₂|（最小）F=|F₁-与较大的力同向
①【巩固题】

两个力F₁=6 N，F₂=8 N，夹角为90°。求合力的大小和方向。

详细解析
F = √(F₁² + F₂²) = √(36 + 64) = √100 = 10 N
tanα = F₂/F₁ = 8/6 = 4/3 ⇒ α ≈ 53°（与F₁方向夹角）
②【巩固题】

将竖直向下的力F=100 N分解为两个分力：一个沿水平方向，另一个与竖直方向成30°角。求两个分力的大小。

详细解析

设水平分力为Fx，倾斜分力为F1（与竖直方向成30°）。

由平行四边形定则：F1的竖直分量 = F1cos30° = F = 100 N

∴ F1 = 100 / cos30° = 100 / (√3/2) ≈ 115.5 N
Fx = F1sin30° = 115.5 × 0.5 ≈ 57.7 N
③【巩固题】

一个物体放在倾角为θ的斜面上，重力为G。将重力分解为沿斜面方向和垂直斜面方向的两个分力，写出表达式。

详细解析
沿斜面方向的分力（下滑力）：G∥ = G sinθ
垂直斜面方向的分力（正压力分量）：G⊥ = G cosθ

注意：这两个分力是重力产生的效果，不是额外的力。在受力分析中，分解了重力就不能再画重力本身。

📖考点2：弹力与摩擦力 
弹力：胡克定律 F = kx（k为劲度系数，x为形变量）。弹力方向

垂直于接触面，绳的拉力沿绳方向。

摩擦力：静摩擦力 0 ≤ f ≤ fmax = μsN；滑动摩擦力 f = μN，方

向与相对运动方向相反。

   【巩固题】1  
 一根轻弹簧，挂上2N重物时伸长了4cm。求弹簧的劲度系数。若改挂5N重物，伸长多少？  
 详细解析F=kx⇒k=F/x=2/0.04=50N/m改挂5N重物：x=F/k=5/50=0.1m=10cm  
 【巩固题】2质量为5kg的物体放在水平面上，物体与地面间的动摩擦因数μ=0.4。求：(1)用15N水平力拉物体时的摩擦力；(2)用25N水平力拉物体时的摩擦力。（g取10m/s²）  
详细解析

最大静摩擦力近似等于滑动摩擦力：

fmax = μN = μmg = 0.4 × 5 × 10 = 20 N
(1) 拉力15 N < 20 N，物体静止，摩擦力为静摩擦力：f = 15
N
(2) 拉力25 N > 20 N，物体运动，摩擦力为滑动摩擦力：f = 20
N
③【巩固题】

质量为m的物体放在倾角为θ的粗糙斜面上静止。求物体受到的摩擦力大小和方向。

详细解析

物体静止，沿斜面方向合力为零：重力沿斜面分力：mg sinθ（向下）摩擦力 f 沿斜面向上，与 mg sinθ 平衡：

f = mg sinθ，方向沿斜面向上。

注意：这是静摩擦力，不是 μmg cosθ（那是最大静摩擦力）。

📋考点3：受力分析步骤 
标准步骤：

(1) 确定研究对象（隔离法或整体法）；

(2) 按"一重二弹三摩擦四其他"顺序分析力；

(3) 画受力示意图；

(4) 建立坐标系（通常沿加速度方向和垂直加速度方向）；

(5) 列平衡方程或牛顿第二定律方程。

①【巩固题】

对静止在粗糙斜面上的物体进行受力分析，画出所有力并说明各力的施力物体。

详细解析

物体受三个力：

1. 重力 G（施力物体：地球），方向竖直向下。

2. 支持力 N（施力物体：斜面），方向垂直于斜面向上。

3. 静摩擦力 f（施力物体：斜面），方向沿斜面向上。

注意：物体静止，三力平衡。不存在"下滑力"，那是重力的分力，不是独立的力。

   【巩固题】2  
   用水平力F推靠在竖直墙上的物体，物体静止。物体质量为m，与墙的动摩擦因数为μ。求墙对物体的摩擦力。  
   详细解析物体受四个力：重力mg（向下）、推力F（水平向右）、墙的支持力N（水平向左）、摩擦力f（竖直向上）。水平方向：N=F竖直方向：物体静止，f=mg注意：此时摩擦力是静摩擦力，大小为mg，不等于μF。μN=只有当物体将要滑动时，最大静摩擦力才等于μF。  
③【巩固题】

A、B两物体叠放在水平面上，用水平力F拉B，整体匀速运动。分别分析A和B的受力。

详细解析
A的受力：重力、B对A的支持力（共2个力）。A在水平方向不受

力，因为A匀速且与B无相对运动趋势。

B的受力：重力、A对B的压力、地面的支持力、拉力F、地面给B

的滑动摩擦力（与F平衡，方向与F相反）。（共5个力）关键：A和B之间没有摩擦力，因为A匀速且无水平外力。

⚖考点4：共点力平衡 

平衡条件：合力为零，即 ΣF = 0（或 ΣFx = 0，ΣFy = 0）。

常用方法：正交分解法、三角形法、相似三角形法。

①【巩固题】

用两根绳子吊起一个重为100 N的物体，两绳与竖直方向的夹角分别为30°和60°。求两绳的拉力。

详细解析

设两绳拉力分别为T₁（与竖直成30°）和T₂（与竖直成60°）。

水平方向：T₁sin30° = T₂sin60°T₁ × 0.5 = T₂ × √3/2 ⇒ T₁ = √3 T₂竖直方向：T₁cos30° + T₂cos60° = 100√3 T₂ × √3/2 + T₂ × 0.5 = 100

1.5 T₂ + 0.5 T₂ = 100 ⇒ 2 T₂ = 100 ⇒ T₂ = 50 N

T₁ = √3 × 50 ≈ 86.6 N
②【巩固题】

长为L的轻绳一端固定，另一端系质量为m的小球。用水平力F缓慢拉小球，使绳与竖直方向成θ角。求拉力F和绳的拉力T。

详细解析

小球受三个力：重力mg、拉力F、绳的拉力T。

水平方向：F = T sinθ竖直方向：T cosθ = mg

解得：T = mg / cosθ，F = mg tanθ
     【巩固题】3质量为m的光滑球放在倾角为θ的斜面和竖直挡板之间。求斜面和挡板对球的支持力。    
     详细解析球受三个力：重力mg、斜面支持力N₁（垂直于斜面）、挡板支持力N₂（水平）。水平方向：N₁sinθ=N₂竖直方向：N₁cosθ=mg解得：N₁cosθ，N₂=mg/=mgtanθ    
⚠ 专题02 易错警示
1. 受力分析遗漏：严格按照"一重二弹三摩擦"顺序，不要漏掉任何一个

力。

2. "下滑力"不存在：它是重力的分力，不是独立的力。受力分析时要么

画重力和两个分力，要么只画重力。

3. 摩擦力方向：与相对运动（趋势）方向相反，不是与运动方向相反。

例如传送带上的物体，摩擦力可能与运动方向相同。

4. 静摩擦力：静摩擦力大小由平衡条件决定，不是 μN。只有最大静摩

擦力才近似等于 μN。

5. 胡克定律：x是形变量（伸长量或压缩量），不是弹簧长度。

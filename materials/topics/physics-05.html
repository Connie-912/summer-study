📖考点1：线速度与角速度 
线速度：v = Δs/Δt = 2πr/T = ωr（方向沿切线方向）
角速度：ω = Δθ/Δt = 2π/T（单位：rad/s）
周期T、频率f、转速n：T = 1/f = 1/n，v = 2πrf，ω = 2πf
物理量符号公式单位
线速度vv=ωr=2πr/Tm/s
角速度ωω=2π/T=v/rrad/s
周期TT=2πr/v=2π/ωs
向心加速度anan=v²/r=ω²rm/s²
①【巩固题】

一质点做匀速圆周运动，半径为2 m，周期为4 s。求：(1) 角速度；(2) 线速度；(3) 向心加速度。

详细解析

(1) 角速度：ω = 2π/T = 2π/4 = π/2 rad/s ≈ 1.57 rad/s

(2) 线速度：v = ωr = (π/2) × 2 = π m/s ≈ 3.14 m/s
(3) 向心加速度：an = v²/r = π²/2 ≈ 4.93 m/s²
②【巩固题】

皮带传动装置中，大轮半径是小轮半径的2倍。大轮边缘的线速度v1与小轮边缘的线速度v2之比是多少？角速度之比呢？

详细解析
皮带传动中，两轮边缘线速度相等（皮带不打滑时）：
v1 : v2 = 1 : 1

由 v = ωr 得 ω = v/r：

ω1 : ω2 = (v/r1) : (v/r2) = r2 : r1 = 1 : 2

大轮角速度是小轮的一半。

钟表上时针、分针、秒针的角速度之比是多少？

详细解析

秒针周期 T秒 = 60 s，分针周期 T分 = 3600 s，时针周期 T时= 43200 sω = 2π/T，所以：ω秒 : ω分 : ω时 = 1/60 : 1/3600 : 1/43200通分：= 720 : 12 : 1

ω秒 : ω分 : ω时 = 720 : 12 : 1
📖考点2：向心力与向心加速度 
向心加速度：an = v²/r = ω²r = 4π²r/T²
向心力：Fn = man = mv²/r = mω²r
关键：向心力是效果力，由某个力或几个力的合力提供，不

是额外的力！

①【巩固题】

质量为0.5 kg的小球，用长1 m的绳子系着在光滑水平面上做匀速圆周运动，线速度为2 m/s。求绳子的拉力。

详细解析

绳子拉力提供向心力：

F = mv²/r = 0.5 × 2² / 1 = 0.5 × 4 / 1 = 2 N

注意：这是拉力，不能说"小球受到向心力2 N"，而应说"绳的拉力2 N提供向心力"。

②【巩固题】

汽车以10 m/s的速度通过半径为50 m的水平弯道。若轮胎与路面间的动摩擦因数为0.4，汽车能否安全通过？（g=10 m/s²）

详细解析

汽车转弯所需向心力：F需 = mv²/r = m × 100/50 = 2m (N)最大静摩擦力（提供向心力）：fmax = μmg = 0.4 × m × 10= 4m (N)F需 = 2m < fmax = 4m能安全通过。

   临界速度：vmax=√(μgr)=√(0.4×10×50)=√200≈14.1m/s  
   【巩固题】3圆锥摆：长为L的绳子一端固定，另一端系质量为m的小球。小球在水平面内做匀速圆周运动，绳子与竖直方向夹角为θ。求小球的角速度ω。  
详细解析

小球受重力mg和绳子拉力T。

竖直方向：T cosθ = mg ⇒ T = mg/cosθ水平方向：T sinθ = mω²r，其中 r = L sinθ(mg/cosθ) × sinθ = mω² × L sinθmg tanθ = mω²L sinθ

g/cosθ = ω²L ⇒ ω = √(g/(L cosθ))
📖考点3：竖直面内的圆周运动 
绳模型（最高点临界速度）：mg = mv²/r ⇒ vmin = √(gr)
杆模型（最高点临界速度）：vmin = 0（杆可以支撑）
拱桥模型：最高点 N = mg - mv²/r，压力小于重力（失重感）
①【巩固题】
 用长0.5m的绳子系一个质量为0.2kg的小球在竖直面内做圆周运动。求小球通过最高点时的最小速度。（g=10m/s²） 
 详细解析最高点绳子拉力恰好为零时，重力提供向心力，速度最小：mg=mvmin²/rvmin=√(gr)=√(10×0.5)=√5≈2.24m/s若速度小于此值，小球在到达最高点前就会脱离圆周轨道。 
②【巩固题】

汽车通过半径为40 m的拱桥顶端时，速度为10 m/s。汽车质量为1000 kg，求此时汽车对桥面的压力。（g=10 m/s²）

详细解析

最高点：mg - N = mv²/rN = mg - mv²/r = 1000 × 10 - 1000 × 100/40 = 10000 -

2500 = 7500 N

由牛顿第三定律，汽车对桥面的压力 = 7500 N。

小于重力（10000 N），处于失重状态。

③【巩固题】

过山车通过半径为20 m的竖直圆环最高点，为保证乘客不脱离座位，过山车通过最高点的最小速度是多少？（g=10 m/s²）

详细解析

最高点，重力与座位支持力提供向心力。临界条件：N = 0mg = mvmin²/r

     vmin=√(gr)=√(10×20)=√200≈14.14m/s这是绳模型的临界速度，低于此速度过山车会脱离轨道。    
   = ⚠专题05易错警示1.向心力不是额外力：向心力是效果力，由重力、弹力、摩擦力等实际力或其合力提供。受力分析时不要画"向心力"！2.匀速圆周运动："匀速"指速率不变，但速度方向时刻变化，是变加速曲线运动。合外力不为零！3.皮带传动：两轮边缘线速度相等（同带），但角速度与半径成反比。同轴传动角速度相等。4.竖直面最高点速度：绳模型最低速度√(gr)，杆模型最低速度v=v0。5.转弯临界速度：水平弯道转弯时，摩擦力提供向心力，vmax=√(μgr)。雨天路面湿滑（μ减小），需减速。

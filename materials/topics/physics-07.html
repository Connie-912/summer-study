⚙考点1：功的计算 
恒力做功：W = F s cosθ
θ = 0°：W = Fs（力与位移同向，正功）
θ = 90°：W = 0（力与位移垂直，不做功）
θ = 180°：W = -Fs（力与位移反向，负功）
总功：W总 = ΣWi = F合 s cosθ
①【巩固题】

用50 N的水平力推一个箱子在水平地面上匀速前进10 m。求：(1)推力做的功；(2) 摩擦力做的功；(3) 合力做的功。

详细解析
(1) 推力做功：WF = Fs cos0° = 50 × 10 × 1 = 500 J
(2) 摩擦力做功：匀速，f = F = 50 N，方向与位移相反
Wf = fs cos180° = 50 × 10 × (-1) = -500 J
(3) 合力做功：W合 = 500 + (-500) = 0 J

合力为零，总功为零，动能不变，符合动能定理。

     【巩固题】2质量为2kg的物体在倾角为37°的斜面上，在平行于斜面向上的拉力F=20N作用下沿斜面向上移动5m。物体与斜面间的动摩擦因    
   数μ=0.25。求：(1)拉力做的功；(2)重力做的功；(3)摩擦力做的功。（sin37°=0.6，cos37°=0.8，g=10m/s²）  
   详细解析(1)拉力做功：WF=20×5=100J(2)重力做功：WG-60=-mgsin37°×s=-2×10×0.6×5=J（重力做负功，因为物体向上运动）(3)摩擦力做功：N=mgcos37°=2×10×0.8=16Nf=μN=0.25×16=4NWf=-fs=-4×5=-20J  
③【巩固题】

关于功，下列说法正确的是：A. 力越大，做功一定越多 B. 位移越大，做功一定越多 C. 摩擦力一定做负功 D. 作用力与反作用力做功的代数和不一定为零

详细解析

A. 错误，功还与位移和夹角有关，力大但位移为零则功为零。

B. 错误，同样与力和夹角有关。

C. 错误，摩擦力可以做正功，例如传送带上的物体被摩擦力加速。

D. 正确，例如两块磁铁相互吸引靠近，作用力与反作用力都做正

功，代数和不为零。

答案：D
⚡考点2：功率 
平均功率：P = W / t
瞬时功率：P = F v cosθ
额定功率：发动机长时间正常工作的最大功率
①【巩固题】

一台起重机在10 s内将质量为500 kg的重物匀速提升20 m。求：(1) 起重机做的功；(2) 起重机的功率。（g=10 m/s²）

详细解析
(1) 匀速提升，拉力 F = mg = 5000 N
W = Fh = 5000 × 20 = 100000 J = 1.0 × 105 J

(2) 平均功率：P = W/t = 100000/10 = 10000 W = 10 kW

②【巩固题】

汽车发动机的额定功率为60 kW，汽车质量为2 t。当汽车以额定功率行驶时，速度为10 m/s时的牵引力为多少？若阻力恒为2000N，最大速度是多少？（g=10 m/s²）

详细解析
v=10 m/s时的牵引力：P = Fv ⇒ F = P/v = 60000/10 =
6000 N
最大速度：当 F = f 时，速度最大
vmax = P/f = 60000/2000 = 30 m/s
   注意：汽车以额定功率行驶时，牵引力随速度增大而减小，速度增大过程是变加速运动。   
   【巩固题】3质量为m的物体从静止开始自由下落，求：(1)t时刻重力的瞬时功率；(2)t时间内重力的平均功率。   
   详细解析(1)瞬时功率：t时刻速度v=gtP瞬=mg×v=mg²t(2)平均功率：t时间内下落高度h=½gt²W=mgh=½mg²t²P均=W/t=½mg²t注意：瞬时功率是平均功率的2倍，因为速度均匀增大。   
📖考点3：动能定理 
动能：Ek = ½mv²
动能定理：W合 = ΔEk = ½mv² - ½mv₀²
适用范围：适用于恒力、变力，直线、曲线运动（力学中最

重要定理之一）

   【巩固题】1  
 质量为2kg的物体，初速度为5m/s，在水平面上受到10N的恒定拉力作用，运动了8m，速度变为9m/s。求摩擦力做的功和摩擦力大小。 
 详细解析由动能定理：W合=ΔEkWF+Wf=½mv²-½mv₀²=½×2×81-½×2×2510×8+Wf=81-25=5680+Wf=56-80=Wf-24J摩擦力大小：f=|Wf|/s=24/8=3N 
②【巩固题】

质量为0.5 kg的小球从离地20 m高处自由落下，落地时速度为18m/s。求空气阻力做的功。（g=10 m/s²）

详细解析

由动能定理：WG + Wf = ΔEkmgh + Wf = ½mv² - 0

0.5×10×20 + Wf = ½×0.5×18²100 + Wf = 0.25×324 = 81

Wf = 81 - 100 = -19 J

空气阻力做负功19 J，消耗了机械能。

 【巩固题】3 
   质量为m的物体，在距地面H高处由静止释放，落入地面下深度为h的沙坑中。求沙坑对物体的平均阻力F。（空气阻力不计）  
   详细解析全程用动能定理：初速度为零，末速度为零=0WG+Wfmg(H+h)-Fh=0F=mg(H+h)/h注意：重力全程做正功（下落H+h），阻力只在沙坑中做负功（位移h）。  
📖考点4：机械能守恒定律 
机械能：E = Ek + Ep（动能 + 势能）
重力势能：Ep = mgh（h为相对于零势能面的高度）
机械能守恒条件：只有重力（或弹力）做功
表达式：½mv₁² + mgh₁ = ½mv₂² + mgh₂
   【巩固题】1小球从高为h的光滑斜面顶端由静止滑下，到达斜面底端时的速度是多少？（用机械能守恒求解）   
   详细解析只有重力做功，机械能守恒。初始：E₁=mgh+0=mgh   
   末态：E₂=0+½mv²mgh=½mv²⇒v=√(2gh)与自由落体结果相同！斜面只是改变了运动方向，不改变末速度大小。   
   【巩固题】2用长L=1m的绳子系着质量为0.2kg的小球，从水平位置释放。求小球到达最低点时的速度和绳子的拉力。（g=10m/s²）   
   详细解析机械能守恒：mgL=½mv²v=√(2gL)=√(2×10×1)=√20≈4.47m/s最低点受力：T-mg=mv²/LT=mg+mv²/L=2+0.2×20/1=2+4=6N   
③【巩固题】

判断下列过程中机械能是否守恒：A. 物体沿粗糙斜面下滑 B. 物体自由下落 C. 跳伞运动员匀速下降 D. 弹簧振子在光滑水平面上振动

详细解析

A. 不守恒，摩擦力做功，机械能转化为内能。

B. 守恒（忽略空气阻力），只有重力做功。

C. 不守恒，匀速下降动能不变，重力势能减小，空气阻力做负

功，机械能减小。

D. 守恒，只有弹簧弹力做功，动能和弹性势能相互转化。

关键：判断是否有除重力（弹力）以外的力做功。

📈考点5：功能关系综合 
核心功能关系：

(1) W合 = ΔEk（动能定理）(2) WG = -ΔEp（重力做功等于重力势能减少量）(3) W其他 = ΔE（除重力、弹力外的力做功等于机械能变化量）(4) 摩擦生热：Q = f · s相对

①【巩固题】

质量为m的物体从高为h的粗糙斜面顶端由静止滑下，到达底端的速度为v。求：(1) 摩擦力做的功；(2) 摩擦产生的热量。（斜面倾角为θ，动摩擦因数为μ）

详细解析
(1) 由动能定理：mgh + Wf = ½mv²
Wf = ½mv² - mgh（负值，因为摩擦力做负功）
(2) 摩擦生热：Q = -Wf = mgh - ½mv²

也可用 Q = f · s = μmg cosθ × (h/sinθ) = μmgh/tanθ两种方法结果应一致。

   【巩固题】2子弹以速度v₀射入固定在水平面上的木块，深度为d。若同样的子弹以速度2v₀射入同样的木块，射入深度为多少？（设阻力恒定）   
   详细解析   

由动能定理，阻力做负功使子弹动能全部转化为内能：

-fd = 0 - ½mv₀² ⇒ fd = ½mv₀²速度变为2v₀时：-fd' = 0 - ½m(2v₀)² = -2mv₀²fd' = 2mv₀² = 4 × (½mv₀²) = 4fd

d' = 4d

速度变为2倍，穿透深度变为4倍（动能与速度平方成正比）。

③【巩固题】

一质量为m的滑块以初速度v₀冲上质量为M的静止木板（放在光滑水平面上），滑块与木板间动摩擦因数为μ，木板足够长。求系统产生的热量。

详细解析
方法一：能量守恒

系统最终以共同速度v运动：mv₀ = (m+M)v ⇒ v = mv₀/(m+M)初始动能：Ek0 = ½mv₀²末动能：Ek = ½(m+M)v² = ½(m+M)[mv₀/(m+M)]² =½m²v₀²/(m+M)

Q = Ek0 - Ek = ½mv₀² - ½m²v₀²/(m+M) = ½mMv₀²/
(m+M)
方法二：Q = f · s相对 = μmg × s相对（需要先求s相对）
⚠ 专题07 易错警示
1. 功的正负：W = Fs cosθ，θ为力与位移的夹角。注意"位移"是物体

的位移，不是力的作用点的位移。

2. 机械能守恒的条件：只有重力（或弹力）做功，不是"只受重力（或

弹力）"。可以有其他力，只要其他力不做功即可。

3. 动能定理：W合是所有力做功的代数和，包括重力做功。不要漏掉重

力！

4. 重力势能：与零势能面的选取有关，但重力势能的变化与零势能面的

选取无关。

5. 摩擦生热：Q = f · s相对，s相对是两接触面间的相对位移，不是对地

位移。

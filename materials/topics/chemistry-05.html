专题05 · 氯及其化合物

氯是典型的非金属元素，Cl2 既是重要的化工原料也是高考常考物质。掌握氯气性质、氯水成分和漂白粉原理是本专题核心。

一、知识精讲
考点1：Cl2 的性质
反应类型化学方程式现象/说明
与金属Na2Na+Cl2=2NaCl剧烈燃烧，黄色火焰，白烟
与金属Fe2Fe+3Cl2=2FeCl3铁丝在中燃Cl2烧，棕褐色烟
与金属CuCu+Cl2=CuCl2铜丝在中燃Cl2烧，棕黄色烟

与水Cl2+H2O⇌HCl+HClO氯水呈浅黄绿色
与NaOHCl2+2NaOH=NaCl+NaClO+H2O实验室吸收尾气
与Ca(OH)22Cl2+2Ca(OH)2=CaCl2+Ca(ClO)2+2H2O制漂白粉

二、典例破题
【典例1】氯水性质

新制氯水与久置氯水相比，下列成分中减少的是（ ）

① Cl2 ② HClO ③ H+ ④ Cl−A. ①②   B. ②③   C. ③④   D. ①④

【答案】A

新制氯水：Cl2+H2O⇌HCl+HClO久置过程中：2HClO = 2HCl + O2↑（见光分解），平衡Cl2+H2O⇌HCl+HClO 不断向右移动。

结果：Cl2 和 HClO 减少，H+ 和 Cl− 增加。久置氯水最终变成稀盐酸。

减少的是 ①②，选 A。

【典例2】漂白粉原理

漂白粉在空气中漂白时发生反应的化学方程式为______，漂白粉久置失效的原因是______（用化学方程式表示）。

【答案与解析】

漂白原理：Ca(ClO)2 + CO2 + H2O = CaCO3↓ + 2HClO（HClO 具有强氧化性，可漂白。）失效原因：2HClO = 2HCl + O2↑（见光分解）久置后 Ca(ClO)2 逐步转化为 CaCO3 和 HCl，最终失去漂白能力。

三、巩固提升   
考点1巩固题   

Q5-1写出 Cl2 与下列物质反应的化学方程式：(1) Fe(2) NaOH 溶液（常温）(3) NaBr 溶液

【答案与解析】

(1) 2Fe + 3Cl2 = 2FeCl3（Fe 被氧化为 +3 价）(2) Cl2 + 2NaOH = NaCl + NaClO + H2O（歧化反应）(3) Cl2 + 2NaBr = 2NaCl + Br2（置换反应，氧化性 Cl2 >Br2）Q5-2将 Cl2 通入含酚酞的 NaOH 溶液中，红色褪去。甲同学认为是 Cl2 与 NaOH反应使溶液碱性减弱所致，乙同学认为是 HClO 的漂白性所致。请设计实验验证。

【答案与解析】

向褪色后的溶液中滴加 NaOH 溶液：若溶液恢复红色，说明甲同学正确（碱性减弱导致褪色）；

若溶液不恢复红色，说明乙同学正确（HClO 漂白性导致褪色，属于永久性褪色）。

实际现象：Cl2+2NaOH=NaCl+NaClO+H2O，生成的 NaClO 水解产生 HClO，具有漂白性，褪色后加碱不恢复红色。故乙同学正确。

考点2 巩固题
 下列物质中，能使干燥的有色布条褪色的是（）干燥的氯水液氯溶液A.Cl2B.C.D.NaClO 
 【答案】BD干燥无漂白性（漂白需要HClO）。A×Cl2氯水中含HClO，具有漂白性。B√液氯是纯净的Cl2（液态），无HClO，无漂白性。C×HClO：ClO−+H2O⇌HClO+OH−，具有漂溶液水解产生D√NaClO白性。 

Q5-4写出漂白粉的有效成分和漂白原理的化学方程式。

【答案与解析】

有效成分：Ca(ClO)2漂白原理：Ca(ClO)2 + CO2 + H2O = CaCO3↓ + 2HClO离子方程式：Ca(ClO)2 + CO2 + H2O = CaCO3↓ + 2H+ +2ClO−（注意 HClO 是弱酸不能拆）实际上离子方程式应为：Ca2+ + 2ClO− + CO2 + H2O =CaCO3↓ + 2HClO

   Q5-5漂白粉长期露置于空气中会失效，写出有关化学方程式。   
   【答案与解析】①Ca(ClO)2+CO2+H2O=CaCO3↓+2HClOO2↑（光照）②2HClO=2HCl+最终转化为和HCl，失去漂白性。Ca(ClO)2CaCO3   
   Q5-6工业上为什么用与消石灰制漂白粉，而不用与NaOH？Cl2Ca(OH)2Cl2   
【答案与解析】

Cl2+2NaOH=NaCl+NaClO+H2O 得到的是 NaClO 溶液（漂白液），不易保存和运输。

2Cl2+2Ca(OH)2=CaCl2+Ca(ClO)2+2H2O 得到固体漂白粉，便于保存和运输。

消石灰 Ca(OH)2 比 NaOH 便宜，成本更低。

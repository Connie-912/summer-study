考点1：正弦定理

✎典例破题

例1：在ΔABC中，A = 60°，B = 45°，a = 6，求b。

解：由正弦定理a/sinA = b/sinB，得 6/sin60° = b/sin45°，∴ b = 6

× sin45° / sin60° = 6 × (√2/2) / (√3/2) = 6√2/√3 = 2√6。

巩固1

在ΔABC中，a = 10，A = 30°，B = 120°，求b。

答案：

b = 10√3。由正弦定理：10/sin30° = b/sin120°，∴ b = 10 ×

sin120° / sin30° = 10 × (√3/2) / (1/2) = 10√3。

巩固2

在ΔABC中，A = 75°，B = 45°，b = 8，求a（sin75° = (√6+√2)/

4）。

答案：

a = 4(√3+1)。由正弦定理：a/sin75° = 8/sin45°，∴ a = 8 × sin75° /

sin45° = 8 × ((√6+√2)/4) / (√2/2) = 8 × (√6+√2)/4 × 2/√2 =

4(√6+√2)/√2 = 4(√3+1)。

巩固3

在ΔABC中，a = 5√3，b = 5，A = 60°，求B。

答案：

B = 30°。由正弦定理：5√3/sin60° = 5/sinB，得 5√3/(√3/2) = 5/

sinB，即 10 = 5/sinB，∴ sinB = 1/2。又a > b，∴ A > B，故B =

30°。

⚠易错警示

·SSA情况（已知两边及其中一边的对角）可能有两解、一解或无解，必须

判断。

·正弦定理中a/sinA = 2R（R为外接圆半径），不是a = sinA。

·角度单位必须统一，不能混用度与弧度。

考点2：余弦定理

✎典例破题

例1：在ΔABC中，a = 7，b = 4√3，c = √13，求最大角。

解：a最大，对应最大角A。cosA = (b　　2+c2−a2)/(2bc) =

(48+13−49)/(2×4√3×√13) = 12/(8√39) = 3/(2√39) =

√39/26。A = arccos(√39/26)。

巩固1

在ΔABC中，a = 3√3，b = 3，C = 30°，求c。

答案：

c = 3。由余弦定理：c2= a2+ b2− 2ab·cosC = 27 + 9 −

2×3√3×3×(√3/2) = 36 − 27 = 9，∴ c = 3。

巩固2

在ΔABC中，a = 5，b = 7，c = 8，判断三角形形状。

答案：

锐角三角形。最大边c = 8，cosC = (a2+b2−c2)/(2ab) =

(25+49−64)/(2×5×7) = 10/70 = 1/7 > 0，故C为锐角，且c最大，∴ 三角

形为锐角三角形。

巩固3

在ΔABC中，b = 6，c = 4，A = 120°，求a。

答案：

a = 2√19。a2= b2+c2−2bc·cosA = 36+16−2×6×4×cos120° =

52−2×24×(−1/2) = 52+24 = 76，∴ a = √76 = 2√19。

⚠易错警示

·余弦定理中符号：cosA = (b2+c2−a2)/(2bc)，分子是"另两边平方和减

对边平方"。

·判断三角形形状时，用余弦值的正负：cosA>0锐角；cosA=0直角；

cosA<0钝角。

·注意防止符号错误，尤其是cos120° = −1/2。

考点3：三角形面积

✎典例破题

例1：在ΔABC中，a = 4，b = 5，C = 60°，求面积S。

解：S = ½ab·sinC = ½ × 4 × 5 × sin60° = 10 × √3/2 = 5√3。

巩固1

在ΔABC中，a = 8，b = 5，C = 30°，求S。

答案：

S = 10。S = ½ab·sinC = ½×8×5×sin30° = 20×1/2 = 10。

巩固2

在ΔABC中，a = 6，b = 8，S = 12√3，求角C。

答案：

C = 60°或C = 120°。S = ½ab·sinC = ½×6×8×sinC = 24sinC =

12√3，∴ sinC = √3/2，故C = 60°或 120°。

巩固3

在ΔABC中，a = 7，b = 8，c = 9，求面积S。

答案：

S = 12√5。p = (7+8+9)/2 = 12，S =

√[12×(12−7)×(12−8)×(12−9)] = √[12×5×4×3] = √720 = 12√5。

⚠易错警示

·S = ½ab·sinC中夹角必须是a和b的夹角C，不能随便用。

·用海伦公式时先求半周长p，不要忘记除以2。

·已知面积反求角时，sinC = k通常有两解（锐角或钝角），要结合其他条

件判断。

✎典例破题

例1：在ΔABC中，acosB = bcosA，判断三角形形状。

解：由正弦定理a=2RsinA, b=2RsinB，代入得 2RsinA·cosB =

2RsinB·cosA，即sinAcosB−sinBcosA=0，sin(A−B)=0，∴

在ΔABC中，sinA : sinB : sinC = 3 : 5 : 7，求最大角。

答案：

C = 120°。由正弦定理a:b:c = sinA:sinB:sinC = 3:5:7，设a=3k,

b=5k, c=7k。cosC = (a2+b2−c2)/(2ab) = (9k2+25k2−49k2)/

(2×3k×5k) = (−15k2)/(30k2) = −1/2，∴ C = 120°。

巩固2

在ΔABC中，a　　2+ b2− c2= ab，求角C。

答案：

C = 60°。由余弦定理cosC = (a2+b2−c2)/(2ab) = ab/(2ab) = 1/2，∴

C = 60°。

巩固3

在ΔABC中，若sin　　2A + sin2B = sin2C，判断三角形形状。

答案：

直角三角形（C = 90°）。由正弦定理得a2+b2=c2，由勾股定理逆定理知

·边化角用a = 2RsinA，角化边用sinA = a/(2R)，注意 2R不要漏掉。

考点5：综合应用

✎典例破题

例1：在ΔABC中，a=2√3，b=2，B=30°，求A和c。

解：由正弦定理sinA = a·sinB/b = 2√3×sin30°/2 = √3/2，∴

A=60°或 120°。若A=60°则C=90°，c = a·sinC/sinA = 2√3×1/

(√3/2) = 4；若A=120°则C=30°，c = 2√3×sin30°/sin120° =

2。本题有两解。

巩固1

在ΔABC中，a=2，b=3，c=4，求cosA + cosB + cosC。

答案：

21/16。cosA = (b2+c2−a2)/(2bc) = (9+16−4)/(2×3×4) = 21/24 =

7/8；cosB = (a2+c2−b2)/(2ac) = (4+16−9)/(2×2×4) = 11/16；

cosC = (a2+b2−c2)/(2ab) = (4+9−16)/(2×2×3) = −3/12 = −1/4。

通分求和：7/8 + 11/16 − 1/4 = 14/16 + 11/16 − 4/16 = 21/16。

巩固2

在ΔABC中，A=60°，b=1，S = √3，求 (a+b+c)/(sinA+sinB+sinC)

的值。

答案：

2√39/3。由S = ½bc·sinA，得 √3 = ½×1×c×(√3/2)，∴ c = 4。由余

弦定理a2= b2+c2−2bc·cosA = 1+16−2×1×4×1/2 = 13，∴ a =

√13。由正弦定理 (a+b+c)/(sinA+sinB+sinC) = 2R = a/sinA = √13/

(√3/2) = 2√39/3。

巩固3

在ΔABC中，S = 15√3，a+b+c = 30，A = 60°，求三边长。

答案：

a=7, b=8, c=15或a=7, b=15, c=8。由S = ½bc·sinA =

½bc·(√3/2) = √3 bc/4 = 15√3，∴ bc = 60。又a2=

b2+c2−2bc·cosA = b2+c2−bc = (b+c)2−3bc = (b+c)2−180。设

b+c = 30−a，则a　　2= (30−a)2−180，解得a = 7。于是b+c = 23，

⚠易错警示

·综合题中正弦定理、余弦定理、面积公式常需联合使用，注意选择合适的

公式。

·SSA情况要特别注意多解讨论，漏解是常见错误。

·解题后要检查结果是否满足三角形的基本条件（两边之和大于第三边

等）。

例1：求过点A(2, 3)且斜率为 4 的直线方程。

解：用点斜式：y − 3 = 4(x − 2)，即y = 4x − 5，化为一般式：4x −

y − 5 = 0。

巩固1

求过点A(1, 2)和B(3, 6)的直线方程。

答案：

y = 2x。k = (6−2)/(3−1) = 2，用点斜式：y−2 = 2(x−1)，即y =

2x。

巩固2

求过点 (2, −1) 且与直线 3x + 4y − 5 = 0平行的直线方程。

答案：

3x + 4y − 2 = 0。平行直线斜率相同，设方程为 3x+4y+C=0，代入点

(2,−1)：6−4+C=0，∴ C=−2。方程为 3x+4y−2=0。

巩固3

求过点 (1, 1) 且与直线 2x − y + 3 = 0垂直的直线方程。

答案：

x + 2y − 3 = 0。原直线斜率k1=2，垂直直线斜率k2=−1/2。用点斜式：

y−1 = (−1/2)(x−1)，即 2y−2 = −x+1，x+2y−3=0。

⚠易错警示

·点斜式y−y0=k(x−x0) 不能表示斜率不存在的直线（竖直线x=x0）。

·两直线平行 ⇔ 斜率相等（k1=k2）或都不存在。

·两直线垂直 ⇔ 斜率乘积为 −1（k1k2=−1）或一个为0一个不存在。

考点2：圆的方程

✎典例破题

例1：求圆心为 (3, −2)，半径为 4 的圆的标准方程。

解：(x−3)2+ (y+2)2= 16。

巩固1

将x2+ y2− 4x + 6y − 3 = 0化为标准方程。

答案：

(x−2)2+ (y+3)2= 16。配方：(x2−4x+4)+(y2+6y+9) = 3+4+9，即

(x−2)2+(y+3)2= 16。圆心 (2,−3)，半径 4。

巩固2

求以A(1, 2)和B(5, 4)为直径端点的圆的方程。

答案：

(x−3)2+(y−3)2= 5。圆心为AB中点 (3,3)，半径r = |AB|/2 =

√[(5−1)2+(4−2)2]/2 = √20/2 = √5。

巩固3

判断方程x2+ y2+ 2x − 4y + 6 = 0是否表示圆。

答案：

不表示圆。D=2, E=−4, F=6。D　　2+E2−4F = 4+16−24 = −4 < 0，不

满足圆的判别条件，方程表示一个虚圆（无实数解）。

⚠易错警示

·一般方程x2+y2+Dx+Ey+F=0表示圆的充要条件：D　　2+E2−4F > 0。

·配方时注意常数项移到右边，记得加到等式两边。

·圆的标准方程中，右边是r2（不是r）。

考点3：位置关系

✎典例破题

例1：判断直线 3x + 4y − 10 = 0与圆x2+ y2= 4 的位置关系。

解：圆心 (0,0)，半径r=2。圆心到直线距离d = |0+0−10|/√(32+42) =

10/5 = 2 = r。d = r，∴直线与圆相切。

巩固1

判断直线y = x + 1与圆x2+ y2= 1 的位置关系。

答案：

相交。直线化为x−y+1=0，圆心 (0,0) 到直线距离d = |0−0+1|/√(1+1)

= 1/√2 < 1 = r，∴相交。

巩固2

求过点P(1, 1)且与圆x2+y2= 2 相切的切线方程。

答案：

x + y = 2。圆心 (0,0)，半径r=√2。设切线为y−1=k(x−1)，即

kx−y−k+1=0。圆心到切线的距离 = |−k+1|/√(k2+1) = √2。解得

k=−1。∴切线为y−1=−1(x−1)，即x+y=2。

巩固3

判断圆C：x2+y2= 4 与圆C：(x−3)2+(y−4)2= 9 的位置关系。

1　　2

答案：

外切。C1圆心 (0,0)，r1=2；C2圆心 (3,4)，r2=3。圆心距d = 5。

r1+r2=5，|r1−r2|=1。d = r1+r2=5，∴ 两圆外切。

⚠易错警示

·直线与圆位置关系：d < r相交，d = r相切，d > r相离。

·两圆位置关系需比较圆心距d与r1+r2和 |r1−r2| 的关系。

·切线问题常用"圆心到直线距离等于半径"来列方程。

一、函数值域求法大全

1. 配方法

适用：二次函数y = ax2+bx+c

步骤：配方 → 确定顶点 → 结合区间

示例：y = x2−4x+5 = (x−2)2+1 ≥ 1

2. 换元法

适用：含根式、分式、复合函数

步骤：设t = g(x) →求t范围 → 求y = f(t)值域

示例：y = x+√(1−x)，令t = √(1−x)

3. 判别式法

适用：y = (ax2+bx+c)/(dx2+ex+f)

步骤：去分母 → 整理为关于x的二次方程 →Δ ≥ 0

注意：检验分母不为零

4. 单调性法

适用：已知单调性的函数

步骤：求定义域 → 判断单调性 → 代入端点

示例：y = 1/x在 [1,3] 上递减，值域 [1/3,1]

5. 图像法

适用：可画出图像的函数

步骤：画图 → 观察y的范围

示例：y = |x−1|+|x−2|的值域为 [1,+∞)

6. 基本不等式法

适用：和积型函数

公式：a+b ≥ 2√(ab)（a,b>0）

示例：y = x+1/x (x>0)的值域 [2,+∞)
